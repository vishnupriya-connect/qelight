Function-Calling and Data Extraction with LLMs

Course Syllabus

What You'll Learn

  • Extend LLMs with custom functionality through function-calling for external function calls.
  • Extract structured data from natural language inputs for practical analysis.
  • Build an end-to-end application to process customer service transcripts using LLMs.

About This Course

This course covers essential skills for creating functionally enhanced applications with LLMs: function-calling and structured data extraction. You’ll learn to define prompts with function instructions, call functions based on natural language inputs, and extract structured data.

  • Learn function-calling basics to call external functions from LLM responses.
  • Implement workflows with multiple function calls, enabling complex agent tasks.
  • Utilize OpenAPI specifications to integrate function calls with web services.
  • Use function-calling to extract structured data from natural language inputs.
  • Build an app that processes customer service transcripts, generates SQL queries, and stores data in a database.

By the end, you’ll be ready to build sophisticated AI-driven agents that automate workflows, enhance recommendation systems, manage data, and support customer feedback analysis.

Course Outline

  • Introduction
    Overview of function-calling and data extraction with LLMs.
  • What is Function Calling
    Introduction to function-calling with LLMs and forming function calls from prompts.
  • Function Calling Variations
    Explore function calling variations including parallel and nested function calls for complex workflows.
  • Interfacing with External Tools
    Building function calls with OpenAPI to integrate with web services.
  • Structured Extraction
    Using function-calling for structured data extraction from unstructured natural language inputs.
  • Applications
    Applying function-calling for real-world applications, from data entry to customer service automation.
  • Course Project: Dialog Processing
    Building a project that processes customer service transcripts, creates SQL calls, and stores data.
  • Conclusion
    Final thoughts and practical next steps.

Who Should Join?

This course is ideal for anyone interested in using function-calling capabilities with LLMs to interface with external tools and extract structured data. Familiarity with LLMs and basic Python knowledge is recommended.
