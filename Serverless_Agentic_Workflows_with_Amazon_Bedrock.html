Serverless Agentic Workflows with Amazon Bedrock

Course Syllabus

What You'll Learn

  • Build and deploy scalable serverless agentic applications with Amazon Bedrock.
  • Integrate tools, code execution, and guardrails for effective agentic action management.
  • Design responsible agents with safeguards against malicious prompts and unintended outputs.

About This Course

This course will teach you to create a serverless agentic application with Amazon Bedrock, using tools, code execution, and guardrails to build efficient and safe agents. You’ll apply these skills to build a customer service bot for a fictional tea mug business, learning how to handle customer queries, retrieve data, and ensure safe and responsible operation.

What You’ll Do

  • Create and deploy an AI agent with Amazon Bedrock, analyzing its actions and results.
  • Connect the agent to CRM services for real-time customer support.
  • Integrate a code interpreter to perform calculations and enhance response accuracy.
  • Implement guardrails to manage sensitive information and inappropriate content.
  • Connect the agent to a knowledge database for answering FAQs and escalation workflows.

Course Outline

  • Introduction: Overview of agentic workflows with Amazon Bedrock.
  • Your first agent with Amazon Bedrock: Build an AI agent and explore its basic functions.
  • Connecting with a CRM: Integrate the agent with CRM systems to access customer information.
  • Performing calculations: Implement a code interpreter for enhanced response accuracy.
  • Guard Rails: Set up safeguards to protect against malicious prompts and secure data.
  • Reading the FAQ manual: Connect to a knowledge repository for customer support.
  • Console Walkthrough: Guide through the Amazon Bedrock interface for configuration and deployment.
  • Conclusion: Recap and insights on building effective agentic workflows.
  • Appendix – Tips and Help: Additional resources and code examples.

Who Should Join?

This course is for anyone with basic Python knowledge who wants to build serverless, agentic applications for business or personal use. Understanding serverless deployment and cloud-based applications is beneficial.
