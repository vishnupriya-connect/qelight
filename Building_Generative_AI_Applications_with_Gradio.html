Building Generative AI Applications with Gradio

Course Syllabus

What You'll Learn

  • Learn how to create and demo machine learning applications quickly.
  • Build your own image generation, image captioning, and text-summarization apps.
  • Share your app with teammates, beta testers, and more on Hugging Face Spaces.

About This Course

Join this short course to learn from Apolinário Passos, ML Art Engineer at Hugging Face, on creating generative AI applications with Gradio. Build interactive apps quickly with minimal code, including:

  • Developing a text summarization app that processes input text and displays the summary.
  • Creating an app for image upload and image-to-text captioning, displaying both image and caption.
  • Building an image generation app that produces images from text input using a diffusion model.
  • Combining captioning and generation: upload an image, caption it, then generate a new image from the caption.
  • Creating an interactive interface to chat with open-source LLMs like Falcon, a top-ranking model.

By the end, you’ll have the knowledge to rapidly build and deploy interactive demos for your projects.

Course Outline

  • Introduction
    Overview of generative AI apps and Gradio's capabilities.
  • NLP Tasks Interface
    Building an interface for NLP tasks like text summarization with code examples.
  • Image Captioning App
    Creating an app that takes an image upload, generates a caption, and displays both the image and caption.
  • Image Generation App
    Building an app that generates images from text input using a diffusion model, with interactive code examples.
  • Describe and Generate Game
    Combining captioning and generation: upload an image, caption it, then generate a new image from the caption.
  • Chat with Any LLM
    Developing an interactive chatbot interface to communicate with open-source LLMs, such as Falcon.
  • Conclusion
    Recap of key concepts and practical tips for deploying Gradio applications.

Who Should Join?

This course is for anyone with basic Python knowledge who wants to quickly build and share machine learning applications using Gradio.
