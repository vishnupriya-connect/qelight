How Business Thinkers Can Start Building AI Plugins With Semantic Kernel

Course Syllabus

What You'll Learn

  • Learn Microsoft’s open-source orchestrator, the Semantic Kernel.
  • Develop business planning and analysis skills while leveraging AI tools.
  • Advance LLM skills using memories, connectors, chains, and more.

About This Course

This course enables business thinkers to harness AI using Microsoft’s open-source orchestrator, Semantic Kernel. Learn how to develop sophisticated business applications using common LLM building blocks, such as memories, connectors, chains, and planners. By the end, you will be able to fully leverage the Semantic Kernel, gaining a systematic approach to integrating and future-proofing your AI tools.

Course Outline

  • Introduction: Overview of course and objectives.
  • Semantic Kernel is Like Your AI Cooking Kitchen: Understanding Semantic Kernel concepts.
  • Cooking Up Flavorful SWOTs with the Kernel: Building strategic tools with Semantic Kernel.
  • Organizing The Tools You Make for Later Reuse: Techniques for tool reuse and organization.
  • Frozen Dinner - The Design Thinking Meal: Using design thinking with AI and Semantic Kernel.
  • Don't Forget to Save the Generated Dripping or The Gravy: Best practices for data storage and retrieval.
  • A Kitchen That Responds to Your "I'm Hungry" is More Than Feasible: Developing responsive AI applications.
  • There's a Fully-Outfitted Professional-Grade Kitchen Ready For You: Using advanced tools and techniques in Semantic Kernel.
  • Conclusion: Course summary and next steps.

Who Should Join?

This course is for anyone interested in Semantic Kernel, especially those with basic Python knowledge and an understanding of APIs. Familiarity with SDKs is helpful but not required.
