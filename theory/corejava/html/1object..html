Objects in Java
  1. Person john = new Person("John", 25); - Create an object with attributes.
  2. john.greet(); - Call a method on an object.
  3. john.age = 26; - Modify an object’s field.
  4. Car myCar = new Car(); - Instantiate a new object.
  5. double speed = myCar.getSpeed(); - Access an object’s method.
  6. Item apple = new Item("Apple"); Item banana = apple; - Reference the same object.
  7. System.gc(); - Request garbage collection (optional).
  8. dog.bark(); - Invoke an object's method.
  9. student.setGrade("A"); - Set an object’s property.
  10. employee.getName(); - Retrieve data from an object.

class Person {
    String name;
    int age;

    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    void greet() {
        System.out.println("Hello, my name is " + name + " and I am " + age + " years old.");
    }
}

// Define the Car class with a method to get the speed
class Car {
    private double speed = 100.5;

    double getSpeed() {
        return speed;
    }
}

// Define the Item class for object reference demonstration
class Item {
    String name;

    Item(String name) {
        this.name = name;
    }
}

// Define the Dog class with a bark method
class Dog {
    void bark() {
        System.out.println("Woof! Woof!");
    }
}

// Define the Student class with a grade setter
class Student {
    private String grade;

    void setGrade(String grade) {
        this.grade = grade;
        System.out.println("Grade set to: " + grade);
    }
}

// Define the Employee class with a name getter
class Employee {
    private String name;

    Employee(String name) {
        this.name = name;
    }

    String getName() {
        return name;
    }
}

// Main class to demonstrate all operations
public class ObjectDemo {
    public static void main(String[] args) {
        // 1. Create an object with attributes
        Person john = new Person("John", 25);
        john.greet(); // Call a method on an object

        // 2. Modify an object’s field
        john.age = 26;
        System.out.println("John's age after modification: " + john.age);

        // 3. Instantiate a new object
        Car myCar = new Car();

        // 4. Access an object’s method
        double speed = myCar.getSpeed();
        System.out.println("Car speed: " + speed);

        // 5. Reference the same object
        Item apple = new Item("Apple");
        Item banana = apple;
        System.out.println("Apple name: " + apple.name + ", Banana name (same object): " + banana.name);

        // 6. Request garbage collection (optional)
        System.gc(); // Requesting garbage collection

        // 7. Invoke an object's method
        Dog dog = new Dog();
        dog.bark();

        // 8. Set an object’s property
        Student student = new Student();
        student.setGrade("A");

        // 9. Retrieve data from an object
        Employee employee = new Employee("Alice");
        String employeeName = employee.getName();
        System.out.println("Employee's name: " + employeeName);
    }
}
Key Points on Class
  1. public class Car { String color; void drive() {} } - Define a class with fields and methods.
  2. Car car = new Car(); - Instantiate an object of a class.
  3. car.color = "Red"; - Assign a value to an object's field.
  4. System.out.println(car.color); - Access an object's field.
  5. public class Car { public Car(String color) { this.color = color; } } - Define a constructor.
  6. Car car = new Car("Blue"); - Create an object using a constructor.
  7. public static int carCount = 0; - Define a static field in a class.
  8. System.out.println(Car.carCount); - Access a static field via the class.
  9. public static void display() { } - Define a static method in a class.
  10. Car.display(); - Call a static method from the class.
Key Points on Inheritance
  1. class Car extends Vehicle {} - Create a class that inherits from another class.
  2. super(color); - Call the superclass constructor from a subclass.
  3. @Override public void drive() { } - Override a superclass method.
  4. super.start(); - Call a superclass method from a subclass.
  5. public class Sedan extends Car {} - Create a multilevel inheritance structure.
  6. class ElectricCar extends Car {} - Define another subclass.
  7. ElectricCar myEV = new ElectricCar(); - Instantiate a subclass object.
  8. myEV.charge(); - Call a method specific to a subclass.
  9. myEV.drive(); - Call an overridden method in the subclass.
  10. System.out.println(myEV instanceof Car); - Check if an object is an instance of a superclass.
Key Points on Interface
  1. public interface Flyable { void fly(); } - Define an interface.
  2. class Bird implements Flyable {} - Implement an interface in a class.
  3. @Override public void fly() { } - Provide implementation for an interface method.
  4. Flyable myBird = new Bird(); - Use an interface as a reference type.
  5. default void glide() { } - Add a default method in an interface.
  6. myBird.glide(); - Call a default method in an interface.
  7. static void inspect() { } - Define a static method in an interface.
  8. Flyable.inspect(); - Call a static method in an interface.
  9. class Plane implements Flyable {} - Implement multiple interfaces.
  10. myPlane.fly(); - Call an interface method implemented in the class.
Key Points on Package
  1. package com.example.vehicle; - Define a package at the top of a Java file.
  2. import com.example.vehicle.Car; - Import a specific class from a package.
  3. import com.example.vehicle.*; - Import all classes from a package.
  4. public class Engine {} - Create a class within a package.
  5. Car car = new Car(); - Use a class from an imported package.
  6. protected int speed; - Use protected access within a package.
  7. public int mileage; - Use public access for external package visibility.
  8. package com.example.access; - Create a different package for classes.
  9. import com.example.vehicle.Engine; - Access a public class from another package.
  10. System.out.println(engine.mileage); - Access a public member in an external package.
