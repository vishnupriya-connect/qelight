Java Key Points

Key Points on Object in Java

  • Definition of Object: An object is an instance of a class, created to use the attributes and behaviors defined in the class. Objects contain state (fields or variables) and behavior (methods).
  • Creating Objects: Objects are created using the new keyword followed by the class constructor. ClassName obj = new ClassName();
  • Accessing Object Members: Object members (fields and methods) are accessed using the dot operator (.). obj.methodName(); or obj.fieldName;
  • Object References: Variables hold references to objects rather than the actual objects. Multiple variables can refer to the same object, allowing shared access and modifications.
  • Garbage Collection: Unused objects are automatically removed by the Garbage Collector in Java. Objects with no references are considered eligible for garbage collection, freeing up memory.

Key Points on Class in Java

  • Definition of Class: A class is a blueprint for creating objects, defining the state (fields) and behavior (methods) of its objects.
  • Class Declaration: Classes are declared using the class keyword, followed by the class name and a pair of curly braces {}. Example: public class ClassName { // fields and methods }
  • Access Modifiers: Classes can use access modifiers such as public, protected, or private to control visibility.
  • Constructors: A class can have constructors to initialize its objects. If no constructor is defined, Java provides a default constructor.
  • Static Members: A class can have static fields and static methods shared across all instances. Example: ClassName.memberName.

Key Points on Inheritance in Java

  • Definition of Inheritance: Inheritance is a mechanism where one class (subclass) inherits fields and methods from another class (superclass). It promotes code reusability.
  • Extending Classes: A class inherits another class by using the extends keyword. Example: class SubClass extends SuperClass {}
  • Types of Inheritance: Java supports single inheritance and multilevel inheritance.
  • Overriding Methods: Subclasses can override superclass methods to provide specific implementations.
  • Using super Keyword: The super keyword is used in a subclass to access the superclass's constructor, fields, or methods.

Key Points on Interface in Java

  • Definition of Interface: An interface is a reference type in Java that can contain only abstract methods (Java 7 and below) or default and static methods (Java 8 and above).
  • Declaring an Interface: Interfaces are declared using the interface keyword. Example: public interface InterfaceName { void methodName(); }
  • Implementing Interfaces: A class uses the implements keyword to adhere to an interface, providing implementations for all its abstract methods.
  • Multiple Inheritance with Interfaces: A class can implement multiple interfaces.
  • Default and Static Methods: Interfaces can include default methods and static methods.

Key Points on Package in Java

  • Definition of Package: A package is a namespace in Java that groups related classes and interfaces.
  • Types of Packages: Built-in packages (e.g., java.util, java.io) and User-defined packages.
  • Creating a Package: A package is created by declaring it at the top of a Java file using the package keyword.
  • Importing Packages: Classes in different packages can be accessed using the import keyword.
  • Access Modifiers with Packages: The public modifier makes classes accessible in any package.

Key Points on Variables in Java

  • Definition of Variables: A variable is a container that holds data of a specific type.
  • Types of Variables: Instance Variables, Static Variables, Local Variables, and Parameters.
  • Variable Declaration and Initialization: Variables must be declared with a data type before use.
  • Scope of Variables: Local scope, Instance scope, and Class scope.
  • Final Variables: Variables declared with the final keyword are constants.

Key Points on Primitive Data Types in Java

  • Definition of Primitive Data Types: Primitive data types are the basic data types in Java.
  • Types of Primitive Data Types: Java has eight primitive data types: byte, short, int, long, float, double, char, and boolean.
  • Memory Allocation: Each primitive data type has a fixed size in memory.
  • Default Values: Primitive types have default values if not explicitly initialized.
  • Wrapper Classes: Each primitive type has a corresponding wrapper class.

Key Points on Arrays in Java

  • Definition of Arrays: An array is a collection of elements of the same data type.
  • Declaring and Initializing Arrays: int[] numbers = {1, 2, 3, 4, 5};
  • Accessing Array Elements: Array elements are accessed using zero-based indexing.
  • Array Length: Arrays have a fixed length accessible via the .length property.
  • Multi-dimensional Arrays: Java supports multi-dimensional arrays, like 2D arrays.
